# Data Analytics Capstone Projects
 
Complete portfolio of 3 end-to-end data analytics projects analyzing 541K+ e-commerce transactions.
 
## Projects
 
**SQL E-Commerce Analytics**
- 15 SQL queries analyzing 541K+ records
- RFM segmentation, customer lifetime value, churn detection
- Key insight: Top 26% customers = 80% revenue, 99.50% repeat rate
- Folder: `01-SQL-Analysis/`
 
**ETL Pipeline (SSIS)**
- Extract 541K+ records from OLE DB, transform with validation, load to SQL Server
- Data type conversions, calculated columns, error handling
- Folder: `02-ETL-Pipeline/`
 
**Power BI Dashboard**
- 3-page interactive dashboard with 8+ KPI cards
- Top 20 Customers/Products tables, pie/line/bar/gauge charts
- Synchronized slicers (Date Range, Customer Segment)
- Folder: `03-PowerBI-Dashboard/`
