# Data Analytics Capstone Projects

Complete portfolio of 3 end-to-end data analytics projects analyzing 541K e-commerce transactions.

## 📊 Projects

### 1️⃣ SQL E-Commerce Analytics
- 15 advanced SQL queries
- RFM segmentation, CLV analysis, churn detection
- Key insight: Top 26% customers = 80% revenue
- 📁 Folder: `01-SQL-Analysis/`

### 2️⃣ ETL Pipeline (SSIS)
- Extract 541K+ records from OLE DB
- Transform with data validation and calculations
- Load to SQL Server
- 📁 Folder: `02-ETL-Pipeline/`

### 3️⃣ Power BI Dashboard
- 3 interactive pages with 8+ KPIs
- Multiple visualizations and charts
- Synchronized slicers for filtering
- 📁 Folder: `03-PowerBI-Dashboard/`
